# dragonboat-online

一款基于 GMS2（GameMaker Studio 2）开发的在线龙舟竞技游戏，支持 2–64 人同房间实时对战。服务器部署于北京，随机分配五张地图，满足 2 人即可开赛。

---

## 特性

- 多人在线：支持 2–64 人同房间  
- 随机地图：共五张地图，由服务器随机分配  
- 自动倒计时：≥2 人入房后开始 30 秒倒计时，结束自动开赛  
- 跨平台客户端：Windows 64 位 & macOS (arm64/x86_64)  
- 分辨率自适应，支持 macOS Retina (2×) 显示  

---

## 操作方式

- 移动：W A S D  
- 点击：鼠标左键  
- 目标：操控龙舟去拿到星星
